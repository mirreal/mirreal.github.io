> 我只是一个微不足道的存在

我忘记了很多事，庆幸的是，记下来的都是值得的事。

很久很久以前，我一直活在多年造就的无聊困境。

有一天，突然想通了很多事，尝试去找一些能提起半点兴趣的事来消磨时光。

而且，小有所成。

可是，在过去的一年多的时间里，我丢掉了很多这期间形成的好的东西。

虽然获得了很多经验，但始终只是循规蹈矩，都没有做出些好玩的东西出来。

比如我的站点几年都没有更新过。

在 Github 几乎没有提交记录。

这让我感到很痛苦。

某种意义上，这种痛苦比杀了我还难受。

现在，我想拿回来。
